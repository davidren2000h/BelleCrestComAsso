# State Removes Single-Family Zoning

The state recently passed **HB 1110**, removing single-family zoning and
replacing it with multifamily zoning.

Key points:

-   Minimum 4 units per lot
-   Bellevue proposals may increase density to 12 units per lot

Possible impacts:

-   Multi-family redevelopment
-   ADUs and DADUs
-   Reduced setbacks
-   Increased building heights

Send comments to: council@bellevuewa.gov
